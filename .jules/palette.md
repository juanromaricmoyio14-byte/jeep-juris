## 2025-02-14 - Localized ARIA Labels
**Learning:** When adding ARIA labels to bilingual apps, the label must be localized based on the active language (e.g., using an existing `isEn` flag or `useTranslation`) to ensure accessibility for all users, not just those speaking the primary language.
**Action:** Always check the surrounding context for localization hooks or flags when adding hardcoded text strings or ARIA labels in bilingual applications.
